Square Game

Skills Needed:

Canvas, Javascript

This game was inspired by playing frogger when I was younger. The goal is to direct the red box to the yellow line using the arrow keys on the keyboard while avoiding the anything that is in blue. Holding down shift gives you a boost to your speed, but not forever. Run out out of boost and your out of luck.

The Square Game was built purely in the HTML5 element canvas using JavaScript to code the logic. I used objects to represent each square (and rectangles in later levels) that had built in collision detection. This makes it very easy to expand the code to build and implement new levels. Currently, no one has reported to have beaten the entire game, which consists of 7 levels.
